# JooqDemo

A taste of JOOQ. Using Spring MVC.

Following to this guide: [Using jOOQ With Spring](http://www.petrikainulainen.net/using-jooq-with-spring/)

### What is working

1. findAll
2. findById
3. delete
4. update
5. add
6. Jackson integration
